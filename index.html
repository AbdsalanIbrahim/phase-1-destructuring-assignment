Welcome to the JavaScript Destructuring Assignment Lab

Introduction

In this lab, you'll practice using destructuring assignment to simplify code that works with arrays, objects, and strings.

Instructions

  • Destructure values from an object and assign them to variables.
  • Destructure values from an array to create variables.
  • Use destructuring with strings to assign parts of a string to variables.

Examples

Here's an example of destructuring an object:

        
          const person = { name: 'Alice', age: 30 };
          const { name, age } = person;
          console.log(name, age); // Outputs: Alice 30

And an example of array destructuring:

        
          const fruits = ['apple', 'banana', 'orange'];
          const [first, second] = fruits;
          console.log(first, second); // Outputs: apple banana
